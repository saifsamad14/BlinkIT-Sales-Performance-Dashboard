# 📊 BlinkIT Sales Performance Dashboard

An interactive **Power BI dashboard** built using **Power BI, SQL, and Excel** to analyze over **8,500 retail sales records**. The dashboard provides actionable insights into sales performance, outlet operations, product categories, and customer purchasing trends through interactive visualizations and KPI tracking.

## 📖 Project Overview

This project focuses on analyzing BlinkIT retail sales data to uncover meaningful business insights. Using Power BI, SQL, and Excel, the dashboard tracks important business metrics and enables users to interactively explore sales performance across different outlet types, locations, product categories, and fat content.

The dashboard is designed to support data-driven decision-making by presenting business KPIs through clean and interactive visualizations.

---

## 🚀 Key Features

- 📈 Interactive Power BI Dashboard
- 📊 KPI Cards for Total Sales, Average Sales, Average Rating & Number of Items
- 🏪 Outlet Performance Analysis
- 📍 Outlet Location & Size Analysis
- 🛒 Product Category Analysis
- 🥛 Fat Content Distribution Analysis
- 📅 Outlet Establishment Trend Analysis
- 🎛 Interactive Filters & Slicers

## 📈 Business Insights

- Tier 3 outlets generated the highest overall sales.
- Medium-sized outlets contributed the largest share of revenue.
- Fruits & Vegetables and Snack Foods were among the highest-selling product categories.
- Regular Fat products slightly outperformed Low Fat products in total sales.
- Outlet performance varied significantly based on establishment year and outlet type.
